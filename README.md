# corpse
Убедитесь, что установлен Python 3.6+.</br>
Git clone</br>
$ git clone https://github.com/Samuraianonweb/corpse</br>
$ cd blockchain</br>

Команды</br>
Создать кошелек</br>
$ python console account create </br>
Запуск майнера </br>
$ python console miner start 3008</br>

Показать кошелек</br>
$ python console account get </br>
